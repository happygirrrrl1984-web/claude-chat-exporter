# claude-chat-exporter
Claude 聊天记录一键导出为PDF
导出效果展示,流程：点击插件-选择「Claude PDF 导出」-「导出为PDF」- PDF已创建并在浏览器打开-⌘+P打印保存即可

适合场景：
历史聊天记录导出再次输入给AI做分析总结；
历史聊天记录导入给AI做背景输入；
很精彩的对话手动本地保存；
适合为了省token和避免上下文污染针对同一个话题换不同窗口多次聊天的用户。

安装方式：保存所有文件至本地文件夹，依照下方截图步骤从chrome打开即可
<img width="738" height="456" alt="image" src="https://github.com/user-attachments/assets/000662b3-ddc3-4819-b7cf-780129e44ede" />

其他说明：
目前支持by对话导出，一次导出对话所有内容，用户和AI的对话都会被导出，相当于历史对话截图保存；
下载后自己用，无数据安全风险，纯本地导出，无隐私安全问题；
目前只有chrome版本；目前仅支持Claude；目前仅支持文字导出，不支持图片和附件导出；
